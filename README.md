# Interface dos Gases Reais
<p>Interface gráfica desenvolvida com o intuito de apresentar os principais tópicos de estudo dos gases reais.</p> <br>
<p>A interface permite a realização de cálculos e a geração de gráficos dos fatores de compressibilidade e das isotermas.</p> <br>
<p>As equações de estado presentes no programa são as seguintes:</p>
<ul>
  <li>Equação de estado de van der Waals;</li>
  <li>Equação de estado de Berthelot;</li>
  <li>Equação de estado de Redlich-Kwong;</li>
  <li>Equação de estado de Dieterici.</li>
</ul>
